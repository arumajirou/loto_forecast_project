# 20. NeuralForecast リソース解析 システムテスト仕様（ブラックボックス）

## 1. 導線
- NFラボに `リソース解析` メニューが表示される。
- メニュー選択で新パネルへ遷移できる。

## 2. 接続/欠損
- DB未接続時にガードメッセージのみ表示される。
- `resources.run` 不在時に安全に中断する。
- `stage_span` 不在時にボトルネックを縮退表示する。

## 3. 典型ケース
- 成功run中心データでKPI/期待値/ランキングが表示される。
- 失敗run混在データでエラー集計/直前イベントが表示される。
- metricありデータで時系列波形が表示される。

## 4. 性能
- `row_limit` 増減で取得件数が制御される。
- 大量データでも画面が極端にフリーズしない。
