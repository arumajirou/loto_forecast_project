# val.md

## 目的
検証対象の操作、保存、生成、表示、異常系を評価し、期待結果との差分を明確にする。

## 対象
- operations dashboard
- DB保存結果
- 生成物
- 実行ログ
- 動的解析ログ

## 制約
- 安全上または破壊的操作で明示確認が必要な場合を除き、追加質問なしで単独判断して進める
- Windows Chrome と Linux fallback を混同しない
- DB差分とファイル差分を証跡として残す

## 成果物
- 評価結果一覧
- pass/fail 判定
- 差分と原因候補
- 再現手順
- 修正提案または修正結果

## 出力形式
1. 評価対象
2. 期待結果
3. 実観測結果
4. 差分
5. 原因候補
6. 対応方針

## 自律完遂条件
- 追加質問なしで完遂する
- 判断可能なことは既存ログ、既存設定、既存コード、既存成果物から判断する
